# ジェイコブの会計学習サイト

Simple Japanese website for posting business Japanese class assignments.

## Files Included
- `index.html` - Homepage
- `jikoshoukai.html` - Self-introduction page
- `jishuu1.html` - Self-study assignment 1
- `jishuu2.html` - Self-study assignment 2
- `jishuu3.html` - Self-study assignment 3
- `jishuu4.html` - Self-study assignment 4
- `style.css` - Stylesheet

## How to Upload to GitHub Pages (FREE)

### Step 1: Create GitHub Account
1. Go to https://github.com and sign up (if you don't have an account)

### Step 2: Create New Repository
1. Click the "+" icon in top right → "New repository"
2. Name it: `jacobkaikei` (or your username)
3. Make it **Public**
4. Click "Create repository"

### Step 3: Upload Files
1. Click "uploading an existing file"
2. Drag all the HTML and CSS files into the box
3. Click "Commit changes"

### Step 4: Enable GitHub Pages
1. Go to repository Settings
2. Click "Pages" in left sidebar
3. Under "Source", select "main" branch
4. Click "Save"
5. Your site will be live at: `https://yourusername.github.io/jacobkaikei/`

### Step 5: Connect Your Domain (Jacobkaikei.com)
1. In GitHub Pages settings, add custom domain: `jacobkaikei.com`
2. In your domain registrar (where you bought the domain):
   - Add a CNAME record pointing to: `yourusername.github.io`
3. Wait 10-30 minutes for DNS to update

## How to Edit Content

### To add content to your pages:
1. Open any `.html` file in a text editor (Notepad, VS Code, etc.)
2. Find the section that says `【ここに内容を書いてください】`
3. Replace it with your content in Japanese
4. Save the file
5. Re-upload to GitHub

### Example edit for jikoshoukai.html:
Replace:
```html
<p>【ここに自己紹介の内容を書いてください】</p>
```

With your actual introduction:
```html
<p>初めまして、ジェイコブと申します。現在、大学で会計学を専攻しています。</p>
```

## Quick Tips
- Keep Japanese at N2 level (not too complex)
- Save files before uploading
- Test locally by opening index.html in your browser
- Upload all files together when making changes
